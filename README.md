# SamsungSettings
삼성 컴퓨터 오류와 해결

### Fn 펑션키가 안먹을 때
```
Fn 키가 안 먹는 경우에는 Samsung Settings를 실행해서 문제를 해결할 수 있다.
간혹 Samsung Settings를 켰을 때 서비스가 동작하지 않는다는 문구가 나온다면 검색창에
서비스 검색 후 Samsung System Service Support를 다시 시작해주면 된다.
```
